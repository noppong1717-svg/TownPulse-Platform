# TownPulse: Full-Stack Digital Platform & CMS

This repository contains the complete source code for **TownPulse**, a comprehensive digital platform showcasing a full-stack architecture, including a robust backend, a web-based CMS, and a mobile application.

## 💻 Tech Stack
* **Web Admin (CMS):** React.js, Tailwind CSS *(Highlighting frontend architecture)*
* **Backend API:** Node.js, Express.js *(RESTful API design)*
* **Database:** PostgreSQL / SQL Schema *(Efficient data management)*
* **Mobile App:** Flutter, Dart

## 📁 Repository Structure
This monorepo is divided into three main services:
* `/backend`: The server-side application providing RESTful APIs and business logic.
* `/web-admin`: A Backoffice / CMS dashboard designed for managing platform data and user workflows.
* `/mobile-app`: A cross-platform mobile application for end-users.

## ✨ Key Responsibilities & Features
* **Database Design:** Architected and optimized relational database schemas (`schema.sql`) to support complex business flows.
* **CMS Development:** Built a responsive and user-friendly backoffice system to manage platform content efficiently.
* **API Integration:** Designed seamless communication between the frontend, mobile app, and backend services.
* **Problem Solving & Tools:** Utilized Git workflow and AI-assisted tools (e.g., Cursor) for code review and optimization.
